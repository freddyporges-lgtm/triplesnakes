# React Exploration

This folder contains the React component-based refactoring exploration of the Triple Snakes application.

## What's Inside

This is a backup of the React architecture exploration that converted the monolithic HTML file into a component-based React application:

- **src/App.tsx** - Main React component
- **src/components/** - Reusable React components (Setup, Scoreboard, TurnEntry, GameLog)
- **src/lib/** - Business logic and utilities (gameLogic, firebase, etc.)
- **src/types/** - TypeScript type definitions
- **tailwind.config.js** - Tailwind CSS configuration
- **src/index.css** - CSS styles

## Key Changes Made

The React exploration included:
- Converting monolithic HTML to component-based architecture
- Visual dice pickers in TurnEntry component
- Tailwind CSS styling
- TypeScript throughout
- Firebase integration

## Status

This exploration has been preserved for future reference. The main codebase has been reverted to the original HTML version while this React work is safely stored here for when you're ready to revisit it.

## How to Use This Later

If you want to return to the React version:
1. You can reference this folder for the component structure
2. The React setup is already in your package.json dependencies
3. The original HTML is preserved in the main branch
